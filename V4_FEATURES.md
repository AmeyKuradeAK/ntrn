# 🚀 NTRN v4.0.0 - Professional AI-Powered Conversion

## 🧠 **AI Provider System**

### **Choose Your AI Provider**
NTRN v4.0 lets you choose between two powerful AI providers:

#### 🥇 **Mistral AI (Recommended)**
- **Best for**: Professional React Native development
- **Model**: mistral-large-latest
- **Features**: Superior code quality, advanced debugging, professional architecture
- **Rate Limit**: 120 requests/min
- **API Key**: Get from [La Plateforme Mistral AI](https://console.mistral.ai/)

#### 🔹 **Gemini 2.0 Flash**
- **Best for**: Fast conversions and simple projects
- **Model**: gemini-2.0-flash
- **Features**: Quick analysis, reliable conversion, good Next.js understanding
- **Rate Limit**: 60 requests/min
- **API Key**: Get from [Google AI Studio](https://aistudio.google.com/)

### **🔑 API Key Setup Process**

1. **First Run**: NTRN will ask you to choose your preferred AI provider
2. **Recommended Choice**: Mistral AI for best results
3. **API Key Entry**: Securely enter your API key when prompted
4. **Storage**: Keys are saved in `.env` file in your project root
5. **Optional**: Configure both providers but use only one at a time

```bash
# Example .env file content
MISTRAL_API_KEY=your_mistral_api_key_here
GEMINI_API_KEY=your_gemini_api_key_here  # Optional
```

### **🔄 Switching Providers**
You can switch between configured providers anytime:

```bash
ntrn --switch-provider
```

## 🚀 **Getting Started**

### **Quick Start**
```bash
# Install latest version
npm install -g ntrn@latest

# Navigate to your Next.js project
cd my-nextjs-app

# Start professional conversion
ntrn
```

### **Command Options**
```bash
ntrn                    # Professional AI conversion (recommended)
ntrn --legacy          # Traditional file-by-file conversion
ntrn --switch-provider # Switch between Mistral AI and Gemini
ntrn --help           # Show help and options
ntrn --version        # Show version
```

### **First-Time Setup**
1. Run `ntrn` in your Next.js project
2. Choose your AI provider (Mistral AI recommended)
3. Enter your API key when prompted
4. Let NTRN analyze and convert your project professionally
